# Figma Import Notes

La version que debe vivir en Figma es esta:

- Logo principal: `logo/raster/kmfx-edge-symbol-glass-original.png`
- Nombre premium: `logo/svg/kmfx-edge-wordmark-glass.svg`
- Logo + nombre: `logo/svg/kmfx-edge-lockup-glass.svg`

## Que borrar del archivo Figma anterior

- Cualquier logo plano blanco/negro.
- Cualquier logo tipo raya.
- Cualquier lockup anterior que use el simbolo plano dentro de un cuadrado.

## Como montarlo

1. Arrastra `kmfx-edge-symbol-glass-original.png` al canvas.
2. Arrastra `kmfx-edge-lockup-glass.svg` para tener el logo con nombre.
3. Usa fondo `#050608`.
4. Mantener el logo principal como imagen/raster para conservar el efecto glass.
5. Usar Figma para ordenar, documentar, prototipar y exportar, no para reconstruir desde cero el efecto 3D.

## Por que no reconstruirlo todo en Figma

Figma puede replicar la estructura y el layout, pero el acabado glass/metalico viene de un render raster. Si se reconstruye solo con vectores planos, se pierde profundidad, brillo frio y presencia premium.
