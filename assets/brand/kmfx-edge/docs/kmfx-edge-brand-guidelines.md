# KMFX Edge Brand Guidelines

## Esencia

KMFX Edge debe sentirse como una herramienta de precision para traders: sobria, tecnologica, disciplinada y orientada a datos. La marca principal es el simbolo glass/metalico con luz fria azul. No usar versiones planas ni marcas antiguas de raya como identidad principal.

## Posicionamiento

KMFX Edge es el edge operativo para traders que necesitan convertir datos, riesgo y disciplina en decisiones consistentes.

Frase base:

> No es solo un porcentaje. Es una mentalidad.

## Logo

El logo se compone de tres piezas:

| Elemento | Nombre | Significado |
| --- | --- | --- |
| Circulo | Punto de inicio | Capital, decision inicial y punto de partida de cada operacion. |
| Linea | Equity | Curva ascendente, balance, gestion y consistencia. |
| Arco | Score | Evaluacion del trader, medicion del rendimiento y gestion del riesgo. |

## Versiones

| Version | Uso |
| --- | --- |
| Simbolo glass original | Logo principal, app icon, favicon, redes, presentaciones y hero visual. |
| Simbolo glass transparente PNG | Dashboard, favicon y lugares donde el fondo cambia entre claro y oscuro. |
| Wordmark glass | Nombre KMFX EDGE con estetica metalica/azul. |
| Lockup glass | Simbolo + nombre para dashboard, web, videos y Figma. |

## Tipografia

Usar Inter como fuente principal en producto, dashboard y web. En piezas de video puede usarse tracking amplio para titulos cortos, manteniendo legibilidad.

Jerarquia recomendada:

- Display: 48-72 px, peso 700.
- H1: 32-44 px, peso 700.
- H2: 22-28 px, peso 650.
- Body: 15-17 px, peso 400.
- Metadata: 12-13 px, peso 500.

## Aplicacion en producto

En el dashboard, el logo debe funcionar como ancla de confianza, no como decoracion. Recomendaciones:

- Sidebar expandida: simbolo glass transparente PNG, sin nombre.
- Sidebar colapsada: simbolo glass transparente PNG.
- Login/auth: simbolo glass con mucho espacio.
- Favicon/PWA: simbolo contenido cuadrado de alto contraste.
- Loading/splash: icono contenido con pulso azul sutil.

## No hacer

- No usar el logo sobre fondos muy ruidosos sin contenedor.
- No aplicar gradientes nuevos a las formas planas.
- No estirar ni comprimir.
- No agregar contornos o sombras extra sobre el logo principal.
- No usar versiones antiguas de raya.
- No separar piezas del logo salvo en animacion/storyboard.
- No poner el logo dentro de una caja negra cuando el dashboard ya tiene su propio fondo.
