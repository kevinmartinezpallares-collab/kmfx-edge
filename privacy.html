Principio de producto

KMFX Edge no necesita tu contraseña del broker para el flujo recomendado. El conector MT5 envía datos operativos mediante una KMFXKey asociada a tu cuenta de KMFX.

1. Datos que recogemos

  • Cuenta KMFX: email, nombre visible, proveedor de login, estado de sesión y permisos.
  • Datos MT5: login de cuenta, servidor, broker, balance, equity, posiciones, operaciones, historial, símbolos, comisiones, swaps, precios y métricas derivadas.
  • Conexión: estado de sincronización, última actividad, KMFXKey en forma segura para validar la cuenta y datos técnicos necesarios para diagnóstico.
  • Billing: plan, estado de suscripción, customer/subscription IDs de Stripe, recibos y eventos de pago necesarios para habilitar acceso.
  • Soporte: mensajes que nos envíes, capturas, logs o datos que compartas voluntariamente para resolver incidencias.

2. Para qué usamos los datos

  • Autenticar usuarios y proteger el acceso al dashboard.
  • Sincronizar cuentas MT5 y calcular métricas de rendimiento, riesgo, ejecución y fondeo.
  • Aplicar límites de plan, permisos y acceso a funciones.
  • Detectar errores de conexión, abuso, rate limits o payloads inválidos.
  • Gestionar suscripciones, facturación, soporte y seguridad del servicio.

3. Proveedores

KMFX Edge puede usar servicios externos para operar el producto: Supabase para autenticación y base de datos, Render para backend, Vercel para hosting web, Cloudflare para proxy y protección de tráfico, Stripe para pagos y Resend u otros proveedores para emails transaccionales.

Estos proveedores solo deben recibir los datos necesarios para prestar su parte del servicio.

4. Datos sensibles y KMFXKeys

Las KMFXKeys se tratan como credenciales. No deben compartirse públicamente. El sistema evita transportarlas en URLs de producción y aplica enmascarado en logs cuando se muestran referencias técnicas.

Si sospechas que una key se ha filtrado, revócala o regenera una nueva desde Cuentas y reinstala o repara el conector con esa key.

5. Conservación y eliminación

Conservamos los datos mientras tu cuenta esté activa o mientras sean necesarios para seguridad, soporte, obligaciones legales o continuidad del producto. Puedes solicitar eliminación o exportación escribiendo a soporte.

Al borrar una conexión MT5, puede dejar de sincronizarse, pero pueden conservarse registros mínimos para auditoría, seguridad, facturación o prevención de abuso.

6. Tus derechos

Según tu ubicación, puedes tener derecho a acceder, corregir, exportar o eliminar datos personales. Para ejercerlos, contacta con soporte indicando el email de tu cuenta KMFX y la solicitud concreta.
